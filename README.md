# Lynk – URL Shortener 🚀

Lynk is a **basic URL shortener** built with **Spring Boot**.  
It takes a long URL and returns a shorter, cleaner link.

## 🛠️ Tech Stack
- Java 17
- Spring Boot 3
- Maven
- H2 Database (for now)

## 🚀 Getting Started
```bash
git clone https://github.com/haarrsshhhh03/lynk-url-shortener.git
cd lynk-url-shortener
mvn spring-boot:run
